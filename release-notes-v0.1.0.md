Gentle Daylight — v0.1.0 (Draft)

A soft light theme designed to reduce eye strain.

What's included
- Theme built from the user's VS Code `workbench.colorCustomizations`.
- Packaged VSIX (attached to this draft) for local install.
- Accessibility audit performed; results available at `accessibility-report.txt` in the repository. The user requested that the chosen colors be preserved, so no color changes were applied.

Installation (local)
- In VS Code: `Extensions: Install from VSIX...` and choose the attached `.vsix` file.
- Or from command line (if `code` CLI is installed):

```bash
code --install-extension gentle-daylight-theme-0.0.1.vsix
```

Notes
- Token colors are not provided explicitly; verify syntax contrast in your environment. See `accessibility-report.txt` for automated findings and recommended adjustments (kept as suggestions only).
